
# 📌
Aplicação clone da NETFLIX simples desenvolvida em React.js utilizando API externa e o banco de dados Firebase

## 📝 Objetivos

> Aperfeiçoar conceitos da biblioteca React
>
> Aprofundar noções básicas de estilização
>
> Experimentação com APIs
> 
> Observar métodos eficientes para melhor interação ao usuário
>

## 🏗️ Estrutura

***src/assets*** - Arquivos diversos (imagens, ícones..)

***src/components*** - Elementos de interface para a aplicação

***src/pages*** - Páginas web a serem trabalhadas

***src/App.jsx*** - Componente funcional que configura a lógica de autenticação e define as rotas da aplicação

***src/main.jsx*** - Definir e renderizar a estrutura base da aplicação e dar início ao React Router, StrictMode, e a montagem do componente raiz 'App'
